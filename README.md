# **CSC500 Module 7 Critical Thinking**

---

## **What This Is:**
<u></u>

This repository contains my Module 7 Critical Thinking assignment for CSC500.

The program uses dictionaries to store course room numbers, instructor names, and meeting times. The user enters a course number, and the program displays the matching course information. The program also validates the user input and keeps prompting until a valid course number is entered.

---

## **What’s Inside:**
<u></u>

- **course_info.py** - Main Python file for the course information program
- **pseudocode.txt** - Pseudocode for the course information program
- **README.md**

---

## **Key Features:**
<u></u>

- Creates a dictionary for course room numbers
- Creates a dictionary for course instructors
- Creates a dictionary for course meeting times
- Gets a course number from the user
- Removes extra spaces from the user input
- Converts the user input to uppercase
- Checks whether the course number is valid
- Displays the room number, instructor, and meeting time for a valid course
- Displays an error message for invalid input
- Keeps prompting until the user enters a valid course number

---

## **How to Run:**
<u></u>

1. Ensure **Python 3** is installed.
2. Open a terminal in the project folder.
3. Run the course information program:
   ```bash
   python3 course_info.py
   ```
4. Enter a course number when prompted.
5. Review the room number, instructor, and meeting time displayed by the program.
